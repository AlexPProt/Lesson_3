## Списки
Для создания ненумерованных спиков необходимо поставить звездочку перед каждым элементом списка
* Элемент 1
* Элемент 2
* элемент 3 

Для создания нумерованного списка необходимо поставить номер с точкой перед элементом списка
1. Элемент1
2. Элемент2
3. Элемент3
